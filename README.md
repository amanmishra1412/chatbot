# AI Chat App (Demo Bot)

A simple AI chat application built with **React** (frontend) and **Express** (backend).  
The app allows users to send messages and receive replies from a **demo bot**.

---

## Features

-   User can send messages via input box
-   Bot replies automatically with demo responses
-   Auto-scroll chat window to show latest messages
-   Clean UI with left-right message alignment

---

## Tech Stack

-   **Frontend:** React, TailwindCSS
-   **Backend:** Node.js, Express, CORS
